========================================================================

一、文件说明
【源码】 存放server.c代码文件
【可执行码】 存放实验release版本的exe文件；hust.jpg、hust.html、hust.png、hust.gif、host.jpg为可访问的测试用例
【图片】 存放实验使用步骤、测试结果图
【注：本部分未使用第三方组件，因此无该目录】

========================================================================

二、开发平台
【操作系统】 Windows 10 专业版
【开发软件】 Visual Studio 2019
【第三方组件】 无
【开发语言】 C/C++

========================================================================

三、运行说明
【1】 首先进入可执行码文件夹，点击server.exe文件
【2】 在控制台输入端口号，例如：80
【3】 在控制台输入监听地址，例如：127.0.0.1
【4】 在控制台输入主目录，例如：.\
【5】 打开浏览器，访问可执行码文件夹中的文件，例如：http://127.0.0.1/test.jpg，其中test.jpg是要访问文件的文件名
          若访问的文件为hust.jpg、hust.html、hust.png、hust.gif，则可以成功访问，返回状态码200
          若访问的文件为host.jpg，则返回状态码403
          若访问的文件为其他不存在的，如test.jpg，则返回状态码400
