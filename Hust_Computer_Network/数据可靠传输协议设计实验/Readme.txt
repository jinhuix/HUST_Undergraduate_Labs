========================================================================

一、文件说明
【源码】 分别存放GBN、SR、TCP三个实验的头文件、源文件
【可执行码】 分别存放GBN、SR、TCP的exe文件；对应检查脚本；运行所需lib库；输入输出文件
【第三方组件】 存放check_win.bat（检查脚本）和netsimlib.lib（第三方库）
【图片】 存放相关实验原理图、测试结果图

========================================================================

二、开发平台
【操作系统】 Windows 10 专业版
【开发软件】 Visual Studio 2019
【第三方组件】 模拟网络环境 API
【开发语言】 C/C++

========================================================================

三、运行说明
【1】 首先进入GBN/SR/TCP对应的可执行码文件夹
【2】 点击 exe文件 运行，或点击 check_win.bat 文件运行检查脚本，可以查看运行是否正确
【3】 在 input.txt 和 output.txt 文件中可以看到输入输出内容
【4】 在 result.txt 文件中可以看到控制台所有内容的重定向输出结果（包含滑动窗口）
【5】 在 GBNOutput.txt/SROutput.txt/TCPOutput.txt 文件中可以看到只有滑动窗口的输出结果
